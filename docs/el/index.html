ενα μοντερνο θεμα για την Forumgreek

Forumactif Edge είναι ένα μοντέρνο θέμα που υποστηρίζει τα κινητά μέσα από τις υπηρεσίες της ForumGreek. Δημιουργημένο και σχεδιασμένο για μία μορφή για όλες τις συσκευές.

Φιλικό για κινητά

Συνδεθείτε και συζητήστε από κάθε συσκευή, από οπουδήποτε, όποτεδήποτε χωρίς κανένα δισταγμό.

Μοντέρνος Σχεδιασμός

Απλός και έξυπνος σχεδιασμός που ταιριάζει σε όλες τις οθόνες.

Εγκατατασταση Forumactif Edge

Για να αποκτήσετε ένα φόρουμ δωρεάν, ForumGreek.

λειτουργιες

Το Forumactif Edge είναι ένα πλήρες θέμα, που δεν δυσκολεύει. Παρέχει αρκετές λειτουργίες για όλους όσους θέλουν να ασχοληθούν με τα φόρουμς.

Ανοιγοκλείσιμα Widgets

Tα widgets είναι σχεδισαμένα έτσι ώστε να είναι πάντα σε μία μπάρα στο πλάι του φόρουμ για την δική σας ευκολία. Έτσι δίνεται σε εσάς μεγαλύτερος χώρος για τα widgets και πιο εύκολα μετάβαση σε αυτά.

Γρήγορη Πλοήγηση

Η μπάρα πλοήγησης που σας ακολουθεί καθώς κινείστε προς τα κάτω/πάνω, έτσι ώστε να μην χρειάζεται να πάτε στο πάνω μέρος της σελίδας για πλοήγηση.

Σταθερός Χρωματισμός Θέματος

Αλλάξτε το θέμα στο πι και φι με τον σταθερό χρωματισμό θέματος, που σας επιτρέπει να αλλάξετε το χρώμα του θέματος σε ότι χρώμα επιθυμείτε!

Σκοτεινή Μορφή

Σβήστε τα φώτα για νυχτερινή πλοήγηση και κάντε τα πράγματα πιο εύκολα για τα μάτια σας. Η σκοτεινή μορφή επιτρέπει σε εσάς και τα μέλη σας να αλλάξετε την φωτεινότητα που έχει το φόρουμ!

Μετρητής Ψήφων

Πατήστε Like ή Dislike σε δημοσιεύσεις χωρίς να χρειαστεί να αλλάξετε σελίδα. Μπορείτε επίσης να δείτε και πόσες ψήφοι έχουν δωθεί σε κάθε δημοσίευση.

Χρωματοποιημένες Δημοσιεύσεις

Οι δημοσιεύσεις είναι χρωματοποιημένες βάσει της ομάδας που ανήκει το μέλος σας, κάνοντας τις δημοσιεύσεις σας πιο εύκολες να αναγνωρίζετε σε ποιά ομάδα ανήκουν.

Εμφανιση ολων των λειτουργιων

θεματα

Το Forumactif Edge παρέχει και φωτεινό και σκοτεινό θέμα, αλλά επίση με πλειθώρα χρωμάτων να επιλέξετε, δίνοντάς σας την δυνατότητα να το προσαρμόσετε όπως σας αρέσει.

Φωτεινό και Σκοτεινό

Επιλέξτε μεταξύ του φωτεινού θέματος ή ένα πιο σκοτεινό θέμα. Δύο σχεδιασμοί για να επιλέξτε αυτό που σας αρέσει!

100+ Χρωματισμοί Θεμάτων

Επιλέξτε μεταξύ εκατοντάδων διαφόρων χρωμάτων για την δημιουργία του φόρουμ σας.

Μετατροπή

Ανεβάστε το επεξεργαζόμενο κείμενο κώδικα και μεταρέψτε το όπως σας αρέσει. Επιπλέον, όλα τα πρότυπα είναι διαθέσιμα για τις ανάγκες σας!

Δεξιά-προς-Αριστερά

Θέματα που υποστηρίζουν την μορφή δεξιά-προς-αριστερά, που κάνει το Forumactif Edge την ιδανικότερη επιλογή για κώδικες που δουλεύουν σε rtl.

εξυπνος πινακας ελεγχου

Το Forumactif Edge έρχεται με έναν έξυπνο αυτοματοποιημένο πίνακα ελέγχου που σας βοηθά με την εγκατάσταση του θέματος αλλά ακι για άλλους σκοπούς, κάνοντας τις εγκαταστήσεις και επεξεργασίες πολυ απλες.

Εγκατάσταση και Απεγκατάσταση

Ο Έξυπνος Πίνακας Ελέγχου σας βοηθά να εγκαταστήσετε αλλά και να απεγκαταστήσετε το Forumactif Edge χωρίς να κάνετε απολύτως τίποτα.

Έλεγχος για Αναβαθμίσεις

Όταν υπάρχει μια νέα έκδοση του Forumactif Edge στο Github, ο έξυπνος πίκανας ελέγχου θα σας ειδοποιήσει και θα σας βοηθήσει να εγκαταστήσετε την νεότερη αναβάθμιση.

Αλλαγή Γλώσσας

Μπορείτε να αλλάξετε την γλώσσα του Forumactif Edge ανά πάσα στιγμή. Απλά επιλέγετε την γλώσσα, κάνοντας κλικ στο "Αλλαγή Γλώσσας" και ο πίνακας ελέγχου θα κάνει την δουλειά για εσάς.

Θέματα και Ρυθμίσεις

Αλλάξτε το πλάτος του φόρουμ, το θέμα κλπ.. γίνεται πολύ πιο εύκολο με τον έξυπνο πίνακα ελέγχου. Απλά επιλέγετε τις ρυθμίσεις που θέλετε, αποθηκεύεται και είστε έτοιμοι!

Εμφάνιση Οδηγού

Πολλαπλες Γλωσσες

Το Forumactif Edge υποστηρίζει πολλαπλές γλώσσες, επιτρέποντάς σας να κάνετε πλοήγηση στην δική σας μητρική γλώσσα.

14 Γλώσσες και μετράμε κι άλλες

Το Forumactif Edge μπορείτε να μεταφραστεί σε πολλαπλές γλώσσες και η υποστήριξη μεγαλώνει χάρη σε ανθρώπους σας εσάς!

Μετάφραση

Αν η γλώσσα σας δεν είναι διαθέσιμη, μπορείτε να την κάνετε διαθέσιμη! Το μόνο που χρειάζεστε είναι σύνδεση στο ίντερνετ και έναν Github λογαριασμό ή στο φόρουμ.

Σχεδιάστηκε από και για την Κοινότητα
